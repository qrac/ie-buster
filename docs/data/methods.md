| Method              | Result                                          |
| ------------------- | ----------------------------------------------- |
| `ieBuster.init()`   | 通常の実行：IE 検出をしてポップアップ表示       |
| `ieBuster.check()`  | IE 検出のみ：付随する処理を同時に行う場合に使う |
| `ieBuster.create()` | ポップアップ表示のみ：IE 検出を含まない         |
| `ieBuster.clear()`  | ポップアップをすべて削除                        |
